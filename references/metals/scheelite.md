# Scheelite (CaWO₄) — Evaluation Notes

*Kept as its own reference file rather than merged into `references/metals/tungsten.md` because scheelite's processing route and typical deposit setting differ enough from wolframite that treating them as interchangeable risks giving wrong flowsheet/cost guidance. Read `tungsten.md` first for the shared market/strategic context, then use this file for scheelite-specific technical points.*

**Deposit type**: predominantly skarn-hosted (contact metamorphic deposits at the margin of intrusions into calcareous host rock) — this is the dominant setting for scheelite specifically, distinct from the quartz-vein/greisen setting more typical of wolframite. Scheelite also occurs in some vein and stratabound settings, but skarn is the setting to default to absent other information.

**Grade reporting**: still reported as % WO₃ (see `tungsten.md`), but confirm scheelite is explicitly identified as the tungsten-bearing mineral (rather than assuming wolframite) — this single fact changes the processing discussion below.

**Processing — this is the key differentiator from wolframite**: scheelite responds poorly to simple gravity concentration compared to wolframite (lower density contrast in some ore types, and scheelite is often finer-grained/more disseminated in skarn ore) and is instead typically recovered via **froth flotation**, often requiring more complex reagent schemes than a typical sulphide flotation circuit, plus final concentrate upgrading (gravity/magnetic/chemical steps) to meet APT-feed specifications. Flotation performance is also sensitive to the presence of other calcium minerals (calcite, fluorite, apatite) common in skarn gangue, which can depress selectivity — check whether a project's metallurgical testwork has specifically addressed this, since it's a common source of underperformance versus bench-scale recovery assumptions.

**Common co-occurring elements**: scheelite skarns not infrequently carry associated molybdenum (as molybdenite) or copper — check for by-product credits the same way you would for any polymetallic skarn system, and don't evaluate the tungsten economics in isolation if co-products are present.

**Common evaluation pitfalls**: applying wolframite-style gravity-circuit capex/opex assumptions to a scheelite skarn project; underestimating flotation selectivity challenges from co-occurring calcium gangue minerals; not checking for by-product Mo/Cu credit in skarn-hosted scheelite systems.
